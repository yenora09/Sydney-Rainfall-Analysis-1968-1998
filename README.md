## 📖 Project Description
This project conducts a comprehensive statistical analysis of monthly rainfall at the Sydney Botanic Gardens over a 31-year period. We utilized Exploratory Data Analysis (EDA) and Non-parametric hypothesis testing (Kruskal-Wallis & Wilcoxon tests) to understand seasonal variations and climate trends.

## 📂 Repository Structure
- `/data`: Contains the raw and cleaned datasets sourced from BOM.
- `/code`: Contains all R scripts used for EDA, visualization, and hypothesis testing.
- `/output`: Contains generated charts (Boxplots, Q-Q plots, Histograms) and statistical result tables.

## 🚀 How to Run the Code
1. Download the dataset from the `/data` folder.
2. Open the R script in `/code` using RStudio.
3. Ensure the `ggplot2` package is installed (`install.packages("ggplot2")`).
4. Run the script to reproduce the plots and test results.
